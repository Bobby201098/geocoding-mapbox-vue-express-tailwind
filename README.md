### Build & Deploy a Geocoding Map App With Mapbox, Vue 3, Express & Tailwind CSS

_Documentation by Romi Julianto_

## Run Server

```bash
- npm run start:dev
```
## Installation

```bash
- npm init
- npm i express dotenv cors axios
- npm i -D nodemon
```
